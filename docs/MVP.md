# MVP 需求文档 - 图片去背工具

**版本**: v1.0  
**日期**: 2026-03-22  
**状态**: 初稿

---

## 1. 产品概述

### 1.1 产品定位
一款极简的图片背景去除在线工具，用户通过浏览器上传图片即可获得去除背景的结果。

### 1.2 核心价值
- **简单**: 无需安装软件，浏览器打开即用
- **快速**: 云端处理，秒级返回结果
- **免费**: 每月 50 次免费额度

### 1.3 目标用户
- 电商卖家（产品图去背）
- 设计师（素材处理）
- 个人用户（证件照、头像）

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持点击选择或拖拽上传 | P0 |
| 图片预览 | 上传前预览原图 | P0 |
| 背景去除 | 调用 Remove.bg API 处理图片 | P0 |
| 结果预览 | 显示去背后的图片 | P0 |
| 图片下载 | 一键下载结果图片 | P0 |

### 2.2 用户流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  打开网站  │ → │ 上传图片  │ → │ 点击处理 │ → │ 预览结果 │ → │ 下载图片 │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

### 2.3 边界情况

- [ ] 文件大小限制（建议 ≤10MB）
- [ ] 文件格式校验（仅支持图片格式）
- [ ] API 错误处理
- [ ] 网络超时处理
- [ ] 免费额度用完提示

## 3. 非功能需求

### 3.1 性能

| 指标 | 目标 |
|------|------|
| 首屏加载 | < 2s |
| API 响应 | < 10s（取决于图片大小） |
| 可用性 | 99.9% |

### 3.2 安全

- [x] 图片不存储到服务器（内存处理）
- [x] API Key 通过环境变量管理
- [x] 无需用户注册

### 3.3 成本

- **Cloudflare Workers**: 免费额度内（10万次/天）
- **Remove.bg**: 免费 50 次/月，超出 $0.017/张

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 原生 HTML/CSS/JS |
| 后端 | Cloudflare Workers |
| API | Remove.bg API |

### 4.2 架构图

```
用户浏览器
    │
    ▼
Cloudflare Workers (CDN)
    │
    ├── / (前端页面)
    │
    └── /api/remove (处理接口)
            │
            ▼
        Remove.bg API
            │
            ▼
        返回结果
```

### 4.3 数据流

1. 用户上传图片 → Workers 接收 FormData
2. Workers 提取图片文件 → 转为 FormData
3. 转发至 Remove.bg API（流式，不落盘）
4. API 返回结果 → Workers 原样返回给浏览器
